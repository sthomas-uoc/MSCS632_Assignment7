# Code for MSCS 632 Assignment 7
Build a stastics calculator in C, OCaml, and Python

## Build and run instructions

### C
The C implementation is in the folder `stats_calc_c`

To build the application run `gcc -o stats_calc stats_calc.c`

To run the application run `./stats_calc <number1> <number2> ...`

### OCaml
The OCaml implementation is in the directory `stats_calc_ocaml`

To build the application, run `ocamlc -o stats_calc stats_calc.ml`

To run the application run `./stats_calc <number1> <number2> ...`

### Python
The Python implementation is in the directory `stats_calc_py`

To run the application, run `python stats_calc.py <number1> <number2> ...`
